A Python tool that analyzes company financial data, calculates key ratios, ranks companies, and generates investment recommendations.

 Features:
- Calculates financial ratios (profit margin, liquidity, leverage, ROA)
- Uses real stock data via yfinance
- Ranks companies based on performance
- Generates BUY / HOLD / RISKY recommendations
- Visualizes results with charts

 Technologies:
- Python
- yfinance
- matplotlib

 How to Run:
1. Install dependencies:
pip install yfinance matplotlib

2. Run:
python analyzer.py
